This is an Employee management system project with a simple CRUD operations.
In order to make it workable properly, you require some steps to follow.

1.) Go to .\DAL\Data\DataContext.cs
2.) Change the connection string to your db in the "OnConfiguring" method
3.) Now Go to .\UI\EMS_APP\src\app
4.) open terminal 
5.) if you do not have Node.js installed in your local machine, please install it first in order to run angular app
6.) run command "npm install"
7.) run command ng serve --watch
8.) run the Dotnet core web API by clicking the green button on Visual studio
